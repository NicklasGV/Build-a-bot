Set up your discord bot application

To Start you need to navigate to Discord Developer Portal from here you need to find the "New Application" button marked on the image to the left.

From here you pick a name for your bot (This can be changed later on)

Create your bot user

Navigate to the "Bot" section located on the left navigations menu.

If a popup appears select "Add bot" to confirm your choice

From here you can personalize your bot however much you want.

You can also generate a bot token used to start your bot Keep this token private and secure

Configure the bot permissions

Keeping on the bot page, locate the "Privileged Gateway Intents" section a bit further down, here you'll need to activate these three intents:

  • PRESENCE INTENT
  • SERVER MEMBERS INTENT
  • MESSAGE CONTENT INTENT

Don't forget to save the changes made.

Start your bot up

Now that you have everything set, you can start making your invitation link for the bot, to be able to add it to a server.

When created the link doesnt contain everything needed for it to work, you'll need to add atleast &scope=bot this is to show the invitation that its for a bot.

And for permissions (Can be added in the Bot section in the bottom) a line like this is added at the back of the url. &permissions=8 (This permission is for Administrator)
